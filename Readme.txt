FOR RUNNING ON LOCAL SYSTEM
1) Install VS Code

2) Install python

3) Run this command in command prompt
pip install gradio langchain transformers bertopic torch langchain-community langchain_groq amadeus langchain_huggingface

4) Create a environment variable 
variable name: OPENAI_API_KEY
variable value: your actual open ai key

5) Download entire repo to local system.

6) open capstone_project.py in VS code and run the py file

7) you should be able to get local host link click on that and good to go





FOR RUNNING ON GOOGLE COLABS
1) Open Capstone_Project_Group_17.ipynb 

2) Create a following keys
variable name: OPENAI_API_KEY
variable value: your actual open ai key

variable name: GROQ_API_KEY
variable value: your actual groq ai key

3) Run the notebook
